# 酒店住房管理系统 python124

## 项目概述

本项目是基于 Python 实现的酒店住房管理系统，采用前后端分离的技术架构。后端使用 Django 框架开发，前端则运用了 Vue.js、CSS3 和 JavaScript 等技术。数据存储采用 MySQL 数据库。

## 技术栈

- 开发语言：Python
- 后端框架：Django
- 前端技术：Vue.js（3.X）、CSS3、JavaScript
- 开发工具：PyCharm、Visual Studio Code、HBuilderX
- 数据库：MySQL 5.7.26

## 功能模块

- 用户管理
- 房间管理
- 预订管理
- 入住/退房管理
- 费用管理
- 报表统计

## 系统角色

- 系统管理员
- 前台服务员
- 客户

## 运行环境

- Python 3.0及以上版本
- MySQL 5.7.26
- 支持主流浏览器的 Web 浏览环境

## 部署步骤

1. 准备 Python 开发环境
2. 安装 MySQL 数据库，并配置
3. 克隆或解压项目代码
4. 安装项目依赖
5. 运行数据库迁移命令，生成数据库表
6. 启动后端服务
7. 前端项目构建与部署
